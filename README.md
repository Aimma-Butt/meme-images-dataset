# meme-images-dataset
This repository contains a curated dataset of approximately **1000 meme images** organized into **29 distinct classes**.  
Each class represents a different sentiment class. 

## Dataset Overview
- Total images: ~1000
- Total classes: 29 (joy, surprise, remorse, relief, realization, pride, optimization, neutral, nervousness, love, grief, sadness, gratitude, fear, excitement, embarrassment, disapproval, disappointment, desire, curiosity, confusion, caring, anger, annoyance, approval, advice, admiration, and amusement)
- Format: JPG / PNG images

## Notes
- Images are collected from publicly available online sources.
- This dataset is intended for **educational and research purposes only**.
- Class distribution may be imbalanced.
